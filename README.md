# Mini Games 🎮

A collection of fun, interactive, and beautifully designed browser-based mini-games built with modern web technologies. This project serves as a unified hub where players can access various classic and creative games, all featuring a cohesive, premium glassmorphic UI and smooth animations.

## 🌟 Features

*   **Modern Unified UI:** A premium dark-themed design system with glassmorphism, responsive CSS Grid layouts, and dynamic gradients.
*   **Fully Playable Games:** Ranging from classic board games to RPGs and logical puzzles.
*   **Responsive Design:** Playable on both desktop and mobile devices.
*   **No Build Tools Required:** Built purely with HTML, CSS (Vanilla), and JavaScript. Just open in your browser and play!

## 🕹️ Included Games

1.  **Tic-Tac-Toe** (`tic-tac-toe.html`)
    *   Classic 3x3 grid game.
    *   **Modes:** Player vs Player, Player vs CPU (Easy/Random), Player vs CPU (Impossible/Minimax Algorithm).
    *   Features a built-in scoreboard and dynamic turn indicators.
2.  **Snake & Ladders** (`snake.html`)
    *   A fully functional digital version of the classic board game.
    *   Supports 1 to 4 players.
    *   Features SVG-drawn snakes and ladders, animated dice rolls, and smooth token movement across a dynamically generated 1-100 board.
    *   Custom player names and token colors.
3.  **Dragon Repeller RPG** (`dragon.html`) *(Formerly game2.html)*
    *   A turn-based mini RPG where you fight monsters, manage gold/health, and buy weapons.
    *   Dynamic visual scenes (Town, Store, Cave, Battle).
    *   Animated health bars, critical hit logic, and an auto-scrolling combat log.
4.  **Burger Boss** (`burger.html`)
    *   A fast-paced clicking game where you must assemble burgers exactly as ordered before time runs out.
    *   Physics-based dropping animations for ingredients.
5.  **Linux Navigation Simulator** (`linux.html`)
    *   An educational terminal-style game designed to teach basic Linux commands.
6.  **Pentest Challenge** (`pentest.html`)
    *   A hacking-themed puzzle game (Probe, exploit, and find the flag).
7.  **Get the Key** (`getthekey.html`)
    *   A puzzle navigation game to collect the key and escape.

## 🚀 How to Play

Since this project uses vanilla web technologies, there is no setup or installation required!

1.  Clone or download this repository to your local machine:
    ```bash
    git clone https://github.com/prateekgit24/minigames.git
    ```
2.  Open the `index.html` file directly in any modern web browser (Chrome, Firefox, Safari, Edge).
3.  Click on any of the game cards to start playing!

## 🛠️ Technologies Used

*   **HTML5:** Semantic structure for all games and the main hub.
*   **CSS3:** Custom properties (variables), CSS Grid, Flexbox, animations (`@keyframes`), and transitions for a polished UI.
*   **JavaScript (ES6):** Game logic, DOM manipulation, turn-based systems, and AI algorithms (Minimax).
*   **Fonts:** Google Fonts ('Outfit' for a modern, clean look).
